## 🚀 Running Locally

### Prerequisites
- [Node.js](https://nodejs.org/) (v18+)
- npm

### Steps

```bash
# 1. Clone the repo
git clone https://github.com/mohanps10/portfolio.git
cd portfolio

# 2. Install dependencies
npm install

# 3. Start dev server (with live reload)
npx vite --host

# 4. Open in browser
Local:   http://localhost:5173/
Network: http://<your-ip>:5173/   ← works on any device on the same WiFi
```
